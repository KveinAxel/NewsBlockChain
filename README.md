## 如何使用

1. `git clone git@github.com:KveinAxel/NewsBlockChain.git`

2. `cd NewsBlockChain\`

3. `mkdir build && cd build`

4. `camke .. `

5. `cmake --build . --target news_block_chain -- -j 4`
